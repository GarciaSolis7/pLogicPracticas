# CAMBIO BILLETE DE $100

##### El código en Java implementa una aplicación gráfica que calcula el cambio en billetes y monedas según una cantidad ingresada por el usuario. 
##### Utiliza Swing para la interfaz gráfica y una lógica de cálculo basada en la división de valores monetarios.

 ### DESCRIPCION DEL CODIGO
##### La aplicación permite ingresar un número (cantidad de billetes de $100)
#####  calcula  el valor total y lo desglosa en denominaciones más pequeñas (billetes de $ 50, $ 20 y monedas de $ 10, $ 5, $ 2, $ 1)

* #### CLASE CAMBIO

##### Define un objeto que representa un billete o moneda. Contiene:

##### Atributos: valor (denominación), cantidad (número de unidades) y tipo (billete o moneda).
##### Métodos:
##### Getters y Setters: Permiten acceder y modificar los valores de los atributos.
##### Constructores: Permiten inicializar objetos Cambio.


```java
package com.mycompany.plf_cambio;


public class Cambio {

    private int valor;
    private int cantidad;
    private String tipo;
    //

    public Cambio( int valor,String tipo) {
        this.valor = valor;
        this.tipo= tipo;
    }

    public Cambio() {
    }
    ///

    public String getTipo() {
        return tipo;
    }

    public void setTipo(String tipo) {
        this.tipo = tipo;
    }

    
   ////

    public int getValor() {
        return valor;
    }

    public void setValor(int valor) {
        this.valor = valor;
    }

    public int getCantidad() {
        return cantidad;
    }

    public void setCantidad(int cantidad) {
        this.cantidad = cantidad;
    }
    
}
```

* #### FRAME PRINCIPAL
##### Es la ventana principal de la aplicación. Contiene:

##### Componentes Gráficos (Swing):
##### Un JSpinner (spnnrNum) para ingresar la cantidad de billetes de $100.
##### Un JButton (btnCambio) para realizar el cálculo.
##### Un JLabel (lblDinero) para mostrar el total en dinero.
##### Un JTextArea (txtALista) para mostrar el desglose del cambio.
##### Lógica de Cálculo:
##### El método calcular() multiplica la cantidad ingresada por 100 y lo desglosa en denominaciones más pequeñas.
##### mostrarLista() genera una cadena con el detalle del cambio.
##### start() inicializa las denominaciones disponibles.

    public void start(){
        String coin="Monedas";
        String bill="Billetes";
        Cambio a= new Cambio( 50, bill);
        cList.add(a);
        //
        Cambio b= new Cambio( 20, bill);
        cList.add(b);
        //
        Cambio c= new Cambio( 10, coin);
        cList.add(c);
        //
        Cambio d= new Cambio( 5, coin);
        cList.add(d);
        //
        Cambio e= new Cambio( 2, coin);
        cList.add(e);
        //
        Cambio f= new Cambio( 1, coin);
        cList.add(f); 
    
    }
    
    public String mostrarLista(){
        String msg="";
        for(Cambio c: cList){
            msg+=c.getCantidad()+" "+c.getTipo()+" de $"+c.getValor()+" \n";
        }
        return msg;
    }
    
    //
    public void calcular (){
        int dinero= numero* 100;
        lblDinero.setText("$ "+dinero);
        //
        for(Cambio c: cList){
            int cantidad= (dinero/ c.getValor());
            c.setCantidad(cantidad);
        }
    }
